# Exercícios de Programação Orientada a Objetos (POO)

Este repositório reúne atividades e exercícios desenvolvidos durante meus estudos no Curso Técnico em Informática do CETEC.

O objetivo deste repositório é armazenar e organizar os exercícios realizados ao longo das aulas, servindo como material de consulta, revisão e acompanhamento da minha evolução na programação orientada a objetos.

## Conteúdos abordados

* Classes e Objetos
* Atributos e Métodos
* Encapsulamento
* Construtores
* Herança
* Polimorfismo
* Abstração
* Relacionamento entre Classes
* Coleções e Estruturas de Dados
* Tratamento de Exceções
* Boas práticas de programação

## Objetivo

Além de praticar os conceitos de Programação Orientada a Objetos, este repositório também contribui para o desenvolvimento da lógica de programação, organização de código e utilização de ferramentas de versionamento.

## Tecnologias utilizadas

* Linguagem de Programação (Java/Python/C# - ajustar conforme necessário)
* Git
* GitHub

## Observação

Os exercícios presentes neste repositório possuem finalidade educacional e foram desenvolvidos como parte das atividades propostas durante o Curso Técnico em Informática do CETEC.
